처음 쓰는 방법

Easy Start
추천 순서 워크스페이스를 불러오면 지금 해야 할 다음 작업을 여기서 바로 안내합니다.
1. 데이터 불러오기 `Dataset Path`에 사진 폴더를 넣고 `Build V1 Workspace`를 누르세요. 원본은 건드리지 않고 작업 폴더만 만듭니다.
2. 실시간 확인 `Start Observe`를 누르면 이미지 수, 오토라벨 수, 세그 마스크 수가 계속 보입니다.
3. 픽셀기술 적용 처음에는 `Apply Recommended`를 쓰세요. 더 강하게 보려면 `Forensic WDR`이나 `Target-Lux`를 누르면 됩니다.
4. 오토라벨링 지금은 `결함 + 조도 이상 영역`까지 같이 제안할 수 있습니다. 선박 사진에서는 이 모드로 먼저 돌리는 게 맞습니다.
5. 검수 proposal이 만들어지면 `Build Review Queue`를 눌러 승인/반려 대상을 정리하세요.
6. 학습/비교 마지막에는 `Plan Training` 또는 `Start Live Training`으로 원본, WDR, 혼합 학습을 비교하세요.
7. 논문형 다중조도 실험 어두운 사진과 밝은 사진을 결합하는 흐름을 한 번에 실행합니다. Dynamic Capture → Forensic WDR → Mertens → DAF → Evidence → Paper Pack 순서입니다.

다중조도 결합 연구 모드

Fusion Research
논문형 실험 흐름 어두운 사진과 밝은 사진을 결합해 결함 증거를 더 잘 보존하는 연구 모드입니다. Dynamic Capture → Forensic WDR → Mertens → DAF → Evidence → Paper Pack 순서로 진행합니다.
WDR 합성 우선 모드
현재는 오토라벨보다 WDR 합성과 중간조도 합성을 먼저 비교하는 것이 좋습니다.
1. 노출 조합 분석 현재 데이터에서 어떤 그룹이 2노출, 3노출 결합에 적합한지 바로 계산합니다.
2. Forensic WDR 보기 좋은 이미지보다 결함 증거 보존을 우선하는 WDR 분기입니다.
3. 고전적 결합 밝은 사진과 어두운 사진을 Mertens로 먼저 결합해 baseline을 만듭니다.
4. 결함 보존 결합 DAF는 결함이 더 잘 보이는 노출의 고주파 정보를 우선 반영합니다.
5. 목표 조도 합성 예: 10lux + 240lux를 100lux 근처 표현으로 만들고 싶을 때 쓰는 분기입니다.
6. 원본+WDR 혼합 학습 원본만이 아니라 raw + forensic_wdr를 같이 학습시켜 일반화 성능을 비교합니다.
7. 논문형 전체 실행 Dynamic Capture부터 Paper Pack까지 한 번에 실행해 논문 초안/그림/실험표 기반을 만듭니다.
